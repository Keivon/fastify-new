# fustily-new
Answer a few prompts, get a custom fastify app in seconds.

## Use as `fastify-new`

### Local development

```bash
npm link
fastify-new my-app
```

### Global install

```bash
npm install -g .
fastify-new my-app
```

### Troubleshooting (Linux/macOS)

```bash
chmod +x cli.js
npm link
```
